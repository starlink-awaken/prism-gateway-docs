## 🎯 核心方案

### 系统架构（极简）

```
触发 → Orchestrator → [3个并行Agent] → [4个串行Agent] → MEMORY → 报告
```

### 7个专业Agent

**并行组（5分钟）：**
1. experience_analyzer - 经验分析
2. knowledge_extractor - 知识萃取
3. pattern_matcher - 模式匹配

**串行组（15分钟）：**
4. sop_generator - SOP生成
5. guideline_updater - 准则更新
6. memory_architect - 记忆架构
7. visualizer - 可视化报告

**主控Agent：**
0. orchestrator - 流程协调

### 三种复盘模式

| 模式 | 耗时 | 适用场景 | 覆盖率 |
|------|------|----------|--------|
| **Quick** | 5分钟 | 小任务、日常迭代 | 80% |
| **Standard** | 25分钟 | 中型项目、阶段完成 | 15% |
| **Deep** | 45分钟 | 重大项目、里程碑 | 5% |

## 💡 核心创新

### 1. 意图识别触发

自动检测项目完成，无需手动触发：
```
"项目做完了" → 自动启动复盘
```

### 2. 多Agent专业分工

7个Agent各司其职，并行+串行高效协作

### 3. 增量知识积累

每次复盘更新MEMORY，知识复利增长：
```
第1次：1个经验
第10次：10个经验 + 3个SOP + 5个模式
第100次：完整知识体系，效率提升10倍
```

### 4. 渐进式披露

Level 1(1分钟) → Level 2(3分钟) → Level 3(10分钟)

避免信息过载

### 5. 历史趋势分析

雷达图历史对比，量化成长轨迹

### 6. 智能模式匹配

自动匹配23个成功模式 + 9个失败模式

### 7. 非侵入式设计

后台自动执行，不阻塞用户工作
