# 报告模块

[← 返回根目录](../CLAUDE.md)

---

## 模块概述

reports/ 目录包含 ReflectGuard 项目的所有报告文档，按类型分类组织。

---

## 目录结构

```
reports/
├── milestone/          # 里程碑报告（Phase、Week 完成报告）
├── task/              # 任务报告（Task 完成报告）
├── quality/           # 质量报告（测试、验证报告）
├── architecture/      # 架构报告（设计文档、重构报告）
├── project/           # 项目管理报告（进度、路线图）
├── operations/        # 运维报告
├── testing/          # 测试报告
├── governance/       # 治理报告
├── github/           # GitHub 相关报告
└── archive/          # 归档报告
```

---

## 报告分类

### 🎯 里程碑报告 (milestone/)
- Phase 完成报告
- Week 完成报告
- Release Notes

### ✅ 任务报告 (task/)
- Task 完成报告
- 功能实现报告

### 🔍 质量报告 (quality/)
- 验证报告
- 测试报告
- 覆盖率报告

### 🏗️ 架构报告 (architecture/)
- Phase 2 架构设计
- 重构报告
- 技术决策文档

### 📊 项目管理报告 (project/)
- 项目进度
- 路线图
- 规划文档
